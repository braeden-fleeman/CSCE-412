Detailed Description

Handles Giving Out Requests.

The Load Balancer has three responsibilities, giving out requests to servers, keeping track of the time, and generating requests. After initial request generation it will randomly generate requests with a 1/35 chance every 10 cycles. The number of new request that can be added is 1 to 2*(# of servers).
